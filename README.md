## 🇺🇸 English Description
Welcome to the **Multiple-dev** shared repository! This is a collaborative space for developers to experiment, share code snippets, and work together on community projects.

### ⚠️ Rules of Engagement
1. **No Overwriting:** Do not modify or delete someone else's code without prior discussion.
2. **Branching:** Please create a separate branch for your features: `git checkout -b feature/your-name-task`.
3. **Commit Messages:** Use clear, descriptive commit messages (e.g., `feat: add login validation`).
4. **Pull Requests:** All changes should be submitted via a Pull Request (PR) for review.
5. **Clean Code:** Follow basic coding standards and keep the structure organized.
